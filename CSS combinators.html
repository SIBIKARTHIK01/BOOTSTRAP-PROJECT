CSS Combinators

A combinators is something that explain the relationship between the selectors.

There are four different combinators in CSS:

1.Descendant selectors (space)

2.Child selectors(>)

3.Adjacent sibling selector(+)

4.General sibling selector(~)



Child selector(>)

The Child selector selects all element that are the Childof a specified element

Adjacent sibling selector(+)

The + selector is used to select the element that is directly after another specific element

General sibling selector(~)

the General sibling selector (~) selects all element that are next sibling of a specified element
